單元 3.1：公文撰寫 Prompt 技巧

📄 下載投影片 PDF

公文是政府機關最常見的文書類型。善用 AI 可以幫助您快速產出公文初稿，但請記得一定要審核內容。

公文 Prompt 的關鍵要素

  • 角色：政府公文撰寫專家
  • 發文對象：明確說明發文給誰
  • 事由/主旨：這份公文要處理什麼事
  • 格式：主旨、說明、辦法（或依據）
  • 語氣：正式公文用語

範例：疫情通報公文

你是政府公文撰寫專家。請根據以下資訊撰寫一份正式公文。 【發文對象】各縣市衛生局 【事由】加強登革熱防治措施 【背景說明】近期南部地區登革熱疫情升溫，需請各縣市加強孳生源清除及衛教宣導 【格式要求】 - 包含：主旨、說明、辦法 - 使用正式公文語氣 - 說明段落以條列方式呈現 - 辦法需明確列出請各單位配合事項

範例對比

✗ 模糊的公文 Prompt
幫我寫一份公文給衛生局
AI 不知道主題、不知道格式要求
✓ 完整的公文 Prompt
你是政府公文撰寫專家。請撰寫發文給各縣市衛生局的函文，主旨為「加強流感疫苗接種作業」，格式包含主旨、說明（含背景與現況）、辦法（請各單位配合事項），使用正式公文語氣。
AI 能產出格式正確的公文初稿

單元 3.2：會議紀錄整理

會議紀錄是許多同仁的日常工作。AI 可以幫助您從會議逐字稿或筆記，快速整理出結構化的會議紀錄。

會議紀錄 Prompt 的關鍵要素

  • 明確的輸出格式：會議資訊、討論重點、決議、待辦
  • 標註重點：權責單位、負責人、期限
  • 提供原始內容：會議逐字稿或筆記

範例：防疫會議紀錄整理

請將以下防疫會議內容整理成結構化的會議紀錄。 【輸出格式】 1. 會議資訊（時間、地點、主持人、出席單位） 2. 疫情現況說明摘要 3. 討論議題與各單位意見 4. 決議事項（含權責單位） 5. 待辦事項（含負責人、期限） 6. 下次會議時間 【注意事項】 - 決議與待辦需明確標註負責單位 - 使用條列式呈現，方便閱讀 - 重點內容使用粗體標示 【會議內容】 （在此貼上會議逐字稿或筆記）

範例對比

✗ 模糊的會議紀錄 Prompt
整理這個會議紀錄 （貼上會議內容）
AI 可能產出格式不一、缺少重要資訊的紀錄
✓ 結構化的會議紀錄 Prompt
請將以下會議內容整理成會議紀錄，格式包含： 1. 會議基本資訊 2. 討論議題摘要 3. 決議事項（標註權責單位） 4. 待辦追蹤（含負責人與期限） （貼上會議內容）
AI 會按照指定格式整理，資訊完整

實際應用情境

情境：整理跨部門防疫協調會議

您參加了一場跨部門的防疫協調會議，手上有會議的速記筆記。會議討論了疫苗配送、衛教宣導分工、經費編列等議題。

使用 AI 的步驟：

  1. 確認筆記中沒有敏感個資
  2. 使用結構化的 Prompt 請 AI 整理
  3. 審核 AI 產出的紀錄是否正確
  4. 補充 AI 可能遺漏的細節
  5. 確認權責單位與期限標註正確

單元 3.3：Email 撰寫技巧

Email 是日常溝通的重要工具。AI 可以幫助您快速撰寫各種類型的 Email。

Email Prompt 的關鍵要素

  • 收件對象：內部同仁？外部單位？民眾？
  • 目的：通知、邀請、回覆、催辦？
  • 語氣：正式、禮貌、簡潔
  • 重點內容：需要傳達的關鍵資訊

常見 Email 類型

會議通知
時間、地點、議程、出席
催辦提醒
事項、期限、聯絡方式
回覆詢問
問題、答覆、後續說明
資料轉發
說明、附件、注意事項

範例：回覆民眾詢問

請撰寫一封回覆民眾的 Email。 【情境】 民眾來信詢問流感疫苗接種相關問題，包含： - 哪裡可以接種？ - 需要付費嗎？ - 接種後有什麼注意事項？ 【Email 要求】 - 語氣：禮貌、親切、專業 - 結構：問候、逐點回覆、結語、署名 - 提供官方資訊查詢管道（如疾管署網站、1922） - 長度適中，不要過長

範例：催辦各單位填報

請撰寫一封催辦 Email。 【情境】 需要催辦各縣市衛生局填報本月疫情監測資料，截止日期為本週五 【Email 要求】 - 收件對象：各縣市衛生局承辦人 - 語氣：正式但不失禮貌 - 內容：填報事項、系統連結、截止時間、聯絡窗口 - 簡潔有力，重點明確
