#### 什麼是生成式 AI (0:40 - 1:30)

**[畫面：動畫展示 AI 運作概念]**

**旁白：**
> 生成式 AI，顧名思義，是能夠「生成」新內容的人工智慧。
>
> 它可以生成文字、圖片、甚至程式碼。
>
> **[畫面：顯示輸入問題 → AI 處理 → 產出回答的流程圖]**
>
> 當您輸入一段文字，也就是我們說的「Prompt」，AI 會分析您的需求，然後生成相對應的回應。
>
> 這個過程就像是與一位知識淵博的助理對話。
>
> 但請記住，它是「助理」，不是「專家」。AI 生成的內容仍需要人工審核確認。

#### AI 能做什麼？ (1:30 - 2:30)

**[畫面：逐一展示各項能力，配合疾管署情境範例]**

**旁白：**

>

>
> **[範例畫面：「請摘要這份疫情週報的重點」]**

>

> 例如：腦力激盪、發想衛教活動主題

---

#### AI 不能做什麼？ (2:30 - 3:30)

**[畫面：警示圖示，強調以下限制；每個限制搭配疾管署真實情境的示例框，紅色警示邊框標示]**

**旁白：**
> 但 AI 也有明確的限制，每一條都與您的工作直接相關：
>
> **AI 可能會「幻覺」**
> 它有時會編造不存在的資訊，講得很有自信但其實是錯的。
>
> **[畫面：紅色警示框示例]**
> **[左側：AI 說「台灣 2023 年登革熱確診共 12,847 例」]**
> **[右側標注：「數字務必至疾管署開放資料平台查證」]**
>
> 特別是數據、法規、政策，一定要經過人工查證，絕不能直接引用。
>
> **AI 無法存取即時資訊**
> 大部分 AI 的知識有截止日期，無法提供最新疫情或法規資訊。
>
> **[畫面：時鐘圖示 + 灰色模糊效果，表示資訊可能已過期]**
>
> **AI 無法取代專業判斷**
> 疫情研判、醫療建議、政策決定，仍需要專業人員的判斷。AI 是輔助，不是決策者。
>
> **AI 不會主動保護隱私**
> 您輸入的內容可能被用於模型訓練，因此絕對不能輸入機敏資料。

---

#### 疾管署的資安原則 (3:30 - 4:25)

**[畫面：設計成可截圖備用的「判斷流程卡」，而非條列文字；旁白結束後顯示「可截圖存檔，隨時參考」提示]**

**旁白：**
> 在疾管署使用 AI，有四條資安原則務必遵守——建議您截圖存下來：
>
> **第一，絕不輸入個人資料**
> 姓名、身分證字號、地址、電話，這些絕對不能輸入 AI。
>
> **第二，不輸入機敏公務資訊**
> 尚未公開的疫情數據、內部策略討論、個案追蹤資料，都不適合使用 AI 處理。
>
> **第三，公開資訊可以使用**
> 已公開的防疫指引、衛教內容、一般性業務文書，可以安全使用。
>
> **第四，產出內容務必審核**
> 任何 AI 生成的內容，在正式使用前都必須經過人工確認。
>
> **[畫面：判斷流程卡動畫，依序顯示]**
> **「這份資料有個人資料？→ 是 → 絕不輸入」**
> **「這份資料是內部機密？→ 是 → 絕不輸入」**
> **「兩者皆否 → 可使用，但產出仍需人工審核」**
>
> 記住這個最簡單的原則：「不確定能不能輸入，那就不要輸入。」

> 限制可以幫助 AI 產出更精準的結果——這就是一個完整的 Prompt。

**[畫面切換：同一份公文，右側出現 AI 協助起草的版本，標注格式已自動套用]**

## 影片嵌入說明

### 兩種嵌入方式

#### 方式一：YouTube 嵌入（推薦）

**優點：**
- 不佔用網站空間
- 自動適應不同網速
- 支援多種畫質
- 觀看數據追蹤
- 免費且穩定

**步驟：**
1. 將影片上傳至 YouTube（可設為「不公開」，只有連結可看）
2. 點擊影片下方的「分享」→「嵌入」
3. 複製 iframe 程式碼
4. 貼入 HTML 檔案中

**HTML 範例：**
```html
<div class="video-container">
    <iframe
        src="https://www.youtube.com/embed/您的影片ID"
        title="課程介紹影片"
        frameborder="0"
        allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen>
    </iframe>
</div>
```

**CSS（已包含在 style.css）：**
```css
.video-container {
    position: relative;
    width: 100%;
    padding-bottom: 56.25%; /* 16:9 比例 */
    margin: 20px 0;
}

.video-container iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border-radius: 8px;
}
```

---

#### 方式二：直接上傳影片檔

**優點：**
- 完全掌控，不依賴外部服務
- 可離線使用

**缺點：**
- 佔用大量空間
- 需要處理不同瀏覽器相容性
- 載入速度可能較慢

**支援格式：**
| 格式 | 副檔名 | 建議 |
|------|--------|------|
| MP4 (H.264) | .mp4 | **最推薦**，相容性最佳 |
| WebM | .webm | 備用格式 |
| OGG | .ogg | 較少使用 |

**HTML 範例：**
```html
<div class="video-container">
    <video controls>
        <source src="videos/module0-intro.mp4" type="video/mp4">
        <source src="videos/module0-intro.webm" type="video/webm">
        您的瀏覽器不支援影片播放。
    </video>
</div>
```

---

### 建議的影片規格

| 項目 | 建議規格 |
|------|----------|
| 解析度 | 1920 x 1080 (Full HD) 或 1280 x 720 (HD) |
| 格式 | MP4 (H.264 編碼) |
| 幀率 | 30 fps |
| 位元率 | 5-10 Mbps |
| 音訊 | AAC, 128-256 kbps |
| 比例 | 16:9 |

### 快速嵌入指南

1. **製作影片**（使用本文件的腳本）
2. **上傳到 YouTube**
   - 登入 YouTube Studio
   - 點擊「建立」→「上傳影片」
   - 可見度設為「不公開」（僅限有連結的人觀看）
3. **取得嵌入碼**
   - 影片頁面 → 分享 → 嵌入
   - 複製 iframe 程式碼
4. **貼入 HTML**
   - 找到對應模組的 `.video-placeholder`
   - 替換為 iframe 程式碼

---

### 注意事項

1. **YouTube 不公開設定**
   - 「不公開」= 只有連結可看，不會出現在搜尋
   - 「私人」= 只有指定帳號可看（不適合教學平台）
   - 「公開」= 所有人可搜尋到

2. **機關網路限制**
   - 如果機關網路封鎖 YouTube，需使用「直接上傳」方式
   - 或建立內部影音伺服器

3. **字幕建議**
   - YouTube 可上傳 .srt 字幕檔
   - 提升可及性與學習效果

4. **影片封面**
   - 建議製作自訂縮圖
   - 包含模組標題與視覺元素

## 影片製作工具推薦

以下工具針對台灣政府機關 e-learning 的特殊需求選用，考量因素包含：IT 採購審查、資料不出境合規性、繁體中文支援、授權成本。

### AI 語音（文字轉語音）

#### 首選：Microsoft Azure Text-to-Speech

- **適用理由**：微軟為台灣政府機關提供 ISO/FedRAMP 認證環境，資料在地化合規性最高；若疾管署已使用 Microsoft 365，可在同一帳號下直接啟用，不需新採購流程。繁體中文語音 `zh-TW-YunJheNeural`（女聲）、`zh-TW-YunFanNeural`（男聲）音質自然，混說英文縮寫（RTFC、AI 等）流暢。
- **費用**：每月前 50 萬字元免費；超出約 NT$120 / 百萬字元
- **注意事項**：需申請 Azure 帳號，建議透過機關的 Microsoft 企業合約申請

#### 次選：ElevenLabs

- **適用理由**：語音品質業界頂尖，情感語調自然；適合需要較高表現力的開場鉤子段落
- **費用**：Starter 方案 US$5/月（30,000 字元），足夠六支影片使用
- **注意事項**：腳本文字需上傳至美國伺服器，請先確認機關資安政策是否允許

---

### 簡報與動畫製作

#### 首選：Microsoft PowerPoint（錄製簡報功能）

- **適用理由**：疾管署若已有 Microsoft 365 授權，PowerPoint 的「錄製投影片放映」功能可直接搭配 AI 語音輸出 MP4，無需採購新軟體，IT 審查障礙為零。支援本方案所有視覺需求：逐行揭露、顏色字卡、分割畫面。
- **費用**：Microsoft 365 授權內含，無額外費用
- **操作流程**：設計投影片 → 「錄製」→「匯出為影片（MP4）」→ 疊加 AI 語音

#### 次選：Canva（免費版）

- **適用理由**：繁體中文介面完整；有豐富的公務風格模板；「簡報轉影片」功能可加入動畫與聲音直接匯出；免費版即可完成本計畫所有影片
- **費用**：免費（基本功能足夠）；Teams 版 NT$450/月（多人協作）
- **注意事項**：影片匯出至 Canva 雲端，確認機關政策後使用
